# Final GDS for Visual Inspection

This folder contains the final GDS generated by LibreLane for the project:

## TinyTapeout GDS Viewer

To inspect the layout visually:

1. Open: https://gds-viewer.tinytapeout.com/
2. Download `tt_um_combolock.gds` from this folder.
3. Drag and drop the GDS file into the viewer.
